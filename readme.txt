This system allows administrators and users to create accounts, and search, borrow and return books.
The code mainly includes Python, Mysql and Qt(provides GUI).

The main Python package is in the file "library":
admin.py----administrator privilege
connect.py----connect Mtsql database
user.py----user access
main.py----main program
gui.py----GUI visibility 

1. Create databases in Mysql.
2. Revise the name of the database in "connect.py" to connect your own database.
3. run "main.py" to use the system.
